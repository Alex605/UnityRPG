# UnityRPG
A single-player role playing game utilizing C# and the Unity game engine. The main character is a wizard who has the
ability to cast spells, fight enemies, level up, and pick up items. Each spell may be leveled up as the player gains
experience, and each spell has its own cast time. The spells are a mixture of point and click spells, as well as skill shots.
The player is only able to attack enemies when they are within his line of sight, and the enemies are only able to attack
the player when he is in their line of sight. Eventually, this game will include things such as NPCs, dungeons, boss fights
and quests.
